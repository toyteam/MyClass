## 简介
班级管理系统是一个以网页为基础，对班级事务进行快捷方便管理的系统。它使用php语言，laravel后台，adminlte前端为开发框架。系统具备问题讨论，数据统计，收班费，发公告，投票，表白墙，资料分享等多种功能，极大方便了学生间的沟通和相关班级事务的处理。
